# OpenLayers Example

OpenLayers example of usage, and script for creation of tiles.

### HOWTO

1. Install Ruby, Paperclip, ImageMagick
2. Put an image with image.jpg name into the data directory
3. Run create_tiles.rb
4. Open index.html
